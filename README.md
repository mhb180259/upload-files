## Description

a project finished by [Nest](https://github.com/nestjs/nest) and [Nest](https://github.com/vitejs/vite)

## Installation

```bash
$ yarn
```

## Running the app

```bash
# run server
$ yarn start:server

# run client
$ yarn start:client

# run build
$ yarn run build:client
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
